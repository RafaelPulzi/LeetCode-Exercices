## 🧠 Metodologia

Para cada exercício, procuro seguir um fluxo de pensamento estruturado:

1. **Entendimento do problema** – leitura atenta e identificação dos inputs/outputs.
2. **Planejamento** – esboço da lógica, talvez com pseudo-código.
3. **Implementação** – codificação da solução em Python.
4. **Análise de complexidade** – reflexão sobre tempo e espaço (Big O).
5. **Otimização** – se possível, busco alternativas mais eficientes.

## 🛠️ Como executar os códigos

1. Clone o repositório:
   ```bash
   git clone https://github.com/RafaelPulzi/LeetCode-Exercices.git
   ```
2. Navegue até a pasta do exercício desejado.
3. Execute com Python:
   ```bash
   python nome_do_arquivo.py
   ```
